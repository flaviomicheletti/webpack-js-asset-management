# Webpack - Asset Management

Inicando com webpack, seguindo o guide abaixo

https://webpack.js.org/guides/asset-management/


## Instalação

    npm i

## Build

    npm run build

## Executando

    node node_modules/http-server/bin/http-server

Abrir em http://localhost:8080/
